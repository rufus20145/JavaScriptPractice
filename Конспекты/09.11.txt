Два основных навыка программиста:
    Два результата:
        Процесс (П)
        Исходный код (ИК)

    Навыки:
        написание исходного кода по процессу: П -> ИК
        чтение исходного кода для составления процесса: ИК -> П (зачастую намного )
    Писать ИК надо так, чтобы его можно было прочитать и понять максимально быстро

Две парадигмы ООП
    
    Simula - ОО надстройка над Algol`om
    С++ - ОО надстройка над С

Сообщение в ООП
    будем говорить, что объект А посылает сообщение объекту Б, если А вызывает у Б какой-либо метод

HTML
    Hyper Text Markup Language
    всё есть элемент - два тега и содержимое между ними - открывается <tag>, закрывается </tag>
    при отсутствии содержимого можно писать только один тег, например <tag/>

CSS
    Cascading Style Sheets
    язык уточнения отображения тегов в HTML
    наибольший приоритет у стиля, который был последним

История JS

    названия: Mocha -> LiveScript -> JavaScript

Структура JS
    состоит из движка, BOM (Browser Object Model), DOM (Document ~ ~)
    чувствителен к регистру

Переменные в JS
        var - локальная переменная в функции, может поломать переменные, объявленные выше

Типы данных в JS
    8 типов
        Number
        String
        Boolean
        Null - пустые объекты (одно значение)
        Nothing - ничего для всего остального (одно значение)
        bigInt - тип для больших целых чисел ( i = 5 - int, i = 5n - bigInt )
        Symbol - хранит уникальные ключи
        Object - объект

Ветвления в JS
    if(условие) {тело}

Циклы в JS 
    for
    while
    foreach

Объекты в JS
    в классе Date месяцы нумеруются с нуля
    цепочка прототипов
    явное и неявное создание объектов
    явно - с помощью функции new
    неявно - var a = [1, 2, 3]; <=> a = new Array(1,2,3);

    функции, которые можно вызвать в виде конструктора, именуем с большой буквы
